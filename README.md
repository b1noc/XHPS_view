# Animation add-on for XHPS toolbox

This add-on creates video files to visualize XHPS simulations.

![Example output](media/example.gif)

# Installation and usage

To use this add-on downlaod the `toolbox` folder and the `animate_orbit.m` file and add them to the Matlab path.
The `animate_orbit.m` file can be used to set specific settings that control the output of the video.
By running `animate_orbit` the video file gets rendered.
